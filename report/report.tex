\chapter{SVM}

\section{SVM简介}

支撑向量机（Support Vector Machine, SVM）是一种二分类模型\cite{r3}，它的基本模型是定义在特征空间上的间隔最大的线性分类器。它与感知机的区别在于 SVM 使分类平面与样本点距离的最大。间隔最大化的问题可形式化为一解凸二次规划的问题，也等价于正则化的合页损失函数的最小化问题。


SVM 还可以利用核技巧，从而对非线性的数据进行分类。
